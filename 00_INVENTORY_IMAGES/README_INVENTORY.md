# CONTENEDOR DE FOTOS DE INVENTARIO AUTOMOTRIZ
## Instrucciones de Operación:
1. Coloca en esta carpeta todas las fotografías reales de los vehículos que deseas procesar para tu catálogo de WhatsApp (Formatos soportados: .jpg, .jpeg, .png).
2. Asegúrate de que los archivos tengan nombres descriptivos (Ejemplo: `ram700_roja.jpg`, `nissan_kicks_gris.png`).
3. Al ejecutar la aplicación en Streamlit, el motor recorrerá automáticamente esta carpeta en lote, aplicará las reglas del manual de marca y te devolverá las imágenes listas.
