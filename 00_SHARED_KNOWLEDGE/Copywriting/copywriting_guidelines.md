# DIRECTRICES DE ESCRITURA Y CONTENIDO
## 1. RESTRICCIÓN DE FORMATO
- Queda estrictamente prohibido el uso de etiquetas HTML o estilos web incrustados. Todo el contenido se entrega en Texto Plano con formato Markdown limpio.

## 2. EVITAR DUPLICIDAD
- Las frases diarias, ganchos (Hooks) y copies publicitarios deben ser dinámicos y completamente diferentes en cada iteración. No se permite la duplicación de textos previos en el feed.
